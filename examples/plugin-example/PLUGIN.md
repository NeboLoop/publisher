# weather-api

A plugin that provides weather data to Nebo agents.

## Bundled Skills

| Skill | Description |
|-------|-------------|
| weather-lookup | Look up current weather and forecasts by location |

## Tools Provided

- `weather.current` — Get current conditions for a location
- `weather.forecast` — Get 7-day forecast
- `weather.alerts` — Get active weather alerts for a region
